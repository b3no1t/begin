# Exercice formulaire HTML

## Les inputs

- Faire une page html.
- Déclarer un `<form>`
- comprenant les inputs et leurs étiquettes respective suivantes:
- - type="text"
- - type="number"
- - type="email"
- - type="date"
- - type="password"
- - type="checkbox"
- - type="radio"
- - type="file"
- - type="submit"
- clore le `<form>`
